# SES worksheet 1

These are the files needed for worksheet 1 - you don't need to change any files so you only need to clone it.

The lab is to get you started connecting to the Olimex board and using Linux and various programs.
To do this worksheet you will need an Olimex board, a USB A to B Cable, an Olimex ARM-USB-Tiny-H JTAG connector.
You will also need access to the Linux machine.

All of this equipement is available in the lab 2q12 or from the help desk for hire.
